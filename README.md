# Steps follow to create chatbot
## 1. conda create -n venv python=3.8 -y
## 2. conda activate venv
## pip install rasa
## rasa init
## rasa add your data to rasa files
## rasa train
## rasa shell --> to chat 
## python app.py --> web app to chat
